# Архитектураи Bozorcha

## Технологияҳо
- **Supabase**: базаи маълумот (PostgreSQL), аутентификатсия, нигоҳдории файлҳо, Edge Functions
- **PostgreSQL**: ҷадвалҳо, индексҳо, RLS, функсияҳо, триггерҳо
- **Edge Functions**: мантиқи сервер (Deno)
- **GitHub**: нигоҳдории рамз ва ҳуҷҷатҳо

## Сохтори базаи маълумот
- **profiles**: маълумоти корбарон (васеъкунии auth.users)
- **categories**: категорияҳои маҳсулот (сохтори дарахтӣ)
- **products**: маҳсулот бо нарх, миқдор ва ҳолат
- **orders**: фармоишҳо бо маблағ ва комиссия
- **order_items**: ашёи фармоиш
- **reviews**: тақризҳо ба маҳсулот
- **payments**: пардохтҳо
- **audit_log**: сабти тағйирот барои пайгирӣ

## Амният
- **RLS (Row Level Security)**: ҳар як ҷадвал дорои политикаҳои дастрасӣ аст
- **Политикаҳо**: барои нақшҳои гуногун (харидор, фурӯшанда, мудир) муайян шудаанд
- **JWT**: барои аутентификатсия (дар версияи оянда)

## Edge Functions
- **search-products**: ҷустуҷӯи маҳсулот бо филтрҳо
- **get-profile**: гирифтани маълумоти корбар
- **create-order**: эҷоди фармоиш (бо кам кардани анбор)
- **payment-webhook**: қабули вебхук аз системаи пардохт

## Индексҳо ва оптимизатсия
- Индексҳо барои суръатбахшии ҷустуҷӯ (B-tree, GIN)
- Индекси мураккаб барои orders бо санаи Тоҷикистон
- Кэш дар Edge Functions барои search-products
- Триггерҳо барои навсозии вақт ва аудит
