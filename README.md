# Bozorcha Core

Маркетплейси рақамӣ барои хариду фурӯш. Лоиҳа дар асоси Supabase ва Edge Functions сохта шудааст.

## Хусусиятҳо
- Ҷустуҷӯи маҳсулот бо филтрҳо
- Эҷоди фармоиш ва идоракунии анбор
- Профили корбар
- Вебхук барои пардохт
- Амният дар сатҳи сатр (RLS)
- Индексҳо барои суръат

## Технологияҳо
- Supabase (PostgreSQL, Auth, Storage, Edge Functions)
- GitHub
