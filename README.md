# Bozorcha Core

Маркетплейси рақамӣ барои хариду фурӯш.

## Технологияҳо
- Supabase (PostgreSQL, Auth, Storage, Edge Functions)
- GitHub

## Насбкунӣ
1. Эҷоди лоиҳа дар Supabase.
2. Иҷрои файлҳои муҳоҷират аз папкаи `supabase/migrations`.
3. Танзими аутентификатсия (Email).
4. Насби Edge Functions.
5. Танзими секретҳо (WEBHOOK_SECRET ва ғ.).
